# 数学問題 抽出・蓄積システム (Math Problem Generator)

AI（Groq API）を活用して、既存の LaTeX ファイルから数学問題を自動抽出し、データベースで管理・再利用するための教材作成支援システムです。

##  主な機能
- **問題解析**: Groq API (Llama 3等) を使用して、TeXファイルから問題を個別に分割・抽出。
- **データベース管理**: 抽出した問題を TinyDB に保存。必要な問題だけをストック可能。
- **ライブラリ機能**: 保存済み問題の閲覧、個別選択、および物理削除。
- **再書き出し**: 選択した問題を再び一括で LaTeX 形式（`.tex`）としてダウンロード。

##  技術スタック
- **Frontend**: Next.js (React)
- **Backend**: Flask (Python)
- **Database**: TinyDB
- **AI**: Groq API
- **Language**: JavaScript, Python

##  セットアップと実行方法

### 1. Python サーバー（バックエンド）の起動
依存ライブラリをインストールし、サーバーを起動します。
```bash
pip install flask flask-cors tinydb
python pdf_server.py
```

### 2.フロントエンドの起動
```bash
npm install
npm run dev
```
### 3.ブラウザでアクセス
http://localhost:3000 にアクセスして利用を開始します。

##  使い方

システムを起動したら、ブラウザで `http://localhost:3000` を開き、以下の手順で操作します。

### 1. 初期設定
- 画面上部の **「APIキー設定」** 欄に、Groq の API キー（`gsk_...`）を入力します。
  - ※ APIキーはブラウザのメモリ内にのみ保持され、サーバーには保存されません。

### 2. 問題の抽出（Home画面）
1. **ファイルの読み込み**: 「TeXファイルを読み込んで解析」セクションで、解析したい `.tex` ファイルを選択します。
2. **AI解析**: AIが自動的に問題を分割・抽出します。完了すると画面に問題カードが表示されます。
3. **選択と保存**: 蓄積したい問題のカードをクリックして選択（枠が緑色になります）し、画面右下の **「DBに保存」** ボタンを押します。
   - 保存された問題は画面から消え、バックエンドのデータベース（TinyDB）に書き込まれます。

### 3. 問題の管理と再利用（Library画面）
1. **ライブラリへ移動**: 画面上部の「Libraryを開く →」リンクをクリックします。
2. **一覧表示**: これまで保存したすべての問題がカード形式で表示されます。
3. **削除**: 不要になった問題は、選択して **「選択した問題を削除」** ボタンでデータベースから完全に消去できます。
4. **TeX書き出し**: 印刷したい問題を選択し、**「選択した問題をTeXでダウンロード」** をクリックすると、それらをまとめた新しい `.tex` ファイルが生成されます。
